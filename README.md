# Sleep Health Analysis: Behavioural and Physiological Driver of Sleep
## Project Overview
- This project explores how demographic, lifestyle and health-related factors influence sleep duration and sleep quality. The goals is to identify key drivers of good vs poor sleep using exploratory data analysis (EDA)

## Objectives
- Understanding Sleep Beahviour across Socio- demographics (Age, Gender, Occupation)
- Understanding Sleep Behaviour across Behaviour/Lifestyle (stress level, physical activity level, number of daily steps)
- Understanding relationships between sleep and health indicators (heart rates, blood pressure, BMI categories)
- Understanding what factors best explain sleep outcomes

## Dataset Features
- Socio-Demographics: age, gender, occupation 
- Sleep metrics: sleep_duration, quality_of sleep, sleep_disorder
- Lifestyle factors: stress_level, physical_activity_level, daily_steps
- Health indicators: heart_rate, blood_pressure, BMI_category

## Key Insights
- Stress is the strongest driver of poor sleep
- Heart rate is strongly linked to sleep quality
- Physical activity activity improves sleep outcomes
- Higher BMI is associated with worse sleep
- Sleep duration and quality are highly correlated
Note: More detailed insights can be seen in the notebook's markdown cells
